#  Automated Data Insight Engine v2.0 (Milestone #40)

##  Novidades da Versão 2.0
- **Pie Chart Integration:** Visualização de distribuição categórica com cálculo automático de percentuais.
- **Smart Validation:** Verificação de integridade de dados antes da geração de gráficos para evitar erros de renderização.
- **Advanced EDA Suite:** Inclui Histogramas, Boxplots, Dispersão, Linhas, Barras e Matrizes de Correlação.

##  Tecnologias
- **Pandas**: Engine de processamento.
- **Seaborn/Matplotlib**: Camada de visualização estatística.
- **Random**: Diversificação de insights em datasets volumosos.

##  Impacto
Uma ferramenta capaz de reduzir o tempo de análise inicial de um cientista de dados de horas para segundos, bastando fornecer o link do dataset.
